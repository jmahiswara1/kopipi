# Kopipi

## Deskripsi
Kopipi adalah sebuah website company profile dan reservasi untuk kedai kopi yang mengusung konsep hangat dan klasik. Website ini dirancang untuk memberikan pengalaman digital yang selaras dengan suasana kedai, menampilkan menu, lokasi cabang, cerita perjalanan bisnis, dan fitur reservasi meja atau ruang meeting. Proyek ini dibangun dengan fokus pada tampilan antarmuka yang modern, animasi yang halus, dan responsivitas penuh untuk berbagai perangkat.

> **Portfolio Note:** Proyek ini adalah sebuah **Studi Kasus & Proyek Portofolio** untuk mendemonstrasikan kemampuan pengembangan *frontend* modern. Ini bukan layanan bisnis operasional yang asli.

## Teknologi yang Digunakan
Proyek ini dikembangkan menggunakan teknologi berikut:

- **Next.js 14**: Framework React utama untuk performa dan optimasi SEO.
- **React**: Library JavaScript untuk membangun antarmuka pengguna.
- **TypeScript**: Superset JavaScript untuk keamanan tipe data dan pengembangan yang lebih terstruktur.
- **Tailwind CSS**: Framework CSS utility-first untuk styling yang cepat dan konsisten.
- **Framer Motion**: Library untuk menangani animasi antarmuka yang kompleks dan interaktif.
- **GSAP (GreenSock Animation Platform)**: Digunakan untuk animasi berbasis scroll yang lebih lanjut (ScrollTrigger).
- **Lucide React**: Koleksi ikon yang ringan dan konsisten.

## Status Pengembangan
Saat ini proyek sedang dalam tahap pengembangan aktif. Fitur-fitur utama seperti halaman beranda, menu, lokasi, dan formulir reservasi telah diimplementasikan. Perbaikan tampilan dan optimasi untuk perangkat seluler terus dilakukan untuk memastikan pengalaman pengguna terbaik.

## Cara Menjalankan
1. Pastikan Node.js telah terinstal.
2. Clone repositori ini.
3. Jalankan `npm install` untuk mengunduh dependensi.
4. Jalankan `npm run dev` untuk memulai server pengembangan lokal.
5. Buka `http://localhost:3000` di browser Anda.
